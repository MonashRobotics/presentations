# Monash Robotics Presentations

Public lab presentations from the Monash Robotics Weekly Meeting and Robotics Reading Group.

## Structure

```
skill-sharing/    ← HPC, tools, workflow tutorials (Weekly Meeting)
paper-reading/    ← paper reading group slides (Reading Group)
```

## Naming convention

`YYYY-MM-DD_Topic_Author.pdf`

## Adding slides

1. Export your slides as PDF
2. Place in the correct folder with the naming convention above
3. Push to `main`
4. Add an entry to the [wiki presentations page](https://monashrobotics.github.io/wiki/presentations/)

Files are served at: `https://monashrobotics.github.io/presentations/`
